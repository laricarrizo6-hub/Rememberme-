Estructura de Carpetas y JSON

Esta app está diseñada para leer archivos estructurados dinámicamente. La estructura de tus archivos JSON y carpetas debe ser la siguiente:

📂 recuerdos-app/ ├── 📁 multimedia/ // ¡Aquí metes TODO junto! │ ├── cristoredentor-1.jpg │ ├── cristoredentor-2.mp4 │ ├── copacabana-1.jpg │ ├── machupicchu-1.jpg │ └── machupicchu-2.jpg ├── 📄 paises.json └── 📄 recuerdos.json

Lógica de Nomenclatura del archivo Multimedia:

  • La carpeta de recuerdos tiene un slug (por ejemplo, cristoredentor).
  • Cualquier archivo dentro de la carpeta multimedia/ que empiece con cristoredentor será cargado automáticamente para esa galería.
  • Si entras a nivel de País (ej. Brasil), la app consolida automáticamente todos los archivos multimedia de todas sus ciudades.
¿Cómo funciona ahora mismo? Hemos cargado un conjunto de datos ficticios dinámicos pero hiper-realistas para que puedas probar la experiencia interactiva, los filtros por tags y la navegación fluida.
